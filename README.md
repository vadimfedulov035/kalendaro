# Internacia Fiksita Kalendaro
## Alternative calendar website in Esperanto
### https://kalendaro.xyz

![cover](logo.jpg)

## Set up website
```shell
./setup.sh
```

You will need to mention:
1) Webhost
2) Email (for certificate)

The script is intended to be executed by root in /root/kalendaro
